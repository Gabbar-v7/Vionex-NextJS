| Suffix                  | Purpose                   | Example                |
| ----------------------- | ------------------------- | ---------------------- |
| `.ts`                   | Regular TypeScript file   | `user-service.ts`      |
| `.tsx`                  | React/JSX TypeScript file | `profile-card.tsx`     |
| `.d.ts`                 | Type declaration file     | `globals.d.ts`         |
| `.types.ts`             | Shared types/interfaces   | `auth.types.ts`        |
| `.type.ts`              | Single main type          | `user.type.ts`         |
| `.interface.ts`         | Interfaces only           | `api.interface.ts`     |
| `.enum.ts`              | Enums                     | `status.enum.ts`       |
| `.constants.ts`         | Constants                 | `routes.constants.ts`  |
| `.utils.ts`             | Utility helpers           | `date.utils.ts`        |
| `.helpers.ts`           | Helper functions          | `auth.helpers.ts`      |
| `.service.ts`           | Service/business logic    | `user.service.ts`      |
| `.controller.ts`        | Controllers               | `auth.controller.ts`   |
| `.repository.ts`        | DB/data access            | `user.repository.ts`   |
| `.hook.ts`              | React hooks               | `use-auth.hook.ts`     |
| `.store.ts`             | State stores              | `cart.store.ts`        |
| `.schema.ts`            | Validation/db schemas     | `user.schema.ts`       |
| `.config.ts`            | Config files              | `vite.config.ts`       |
| `.test.ts` / `.spec.ts` | Tests                     | `auth.service.test.ts` |
